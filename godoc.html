    func Decrypt

    func Decrypt(selector, privkeypath string, in []byte) (out []byte, err error)

    Decrypt will decrypt the data in 'in' and return it in 'out', given the path to a PEM-encoded RSA private key file and a selector, which must be the same used for encryption

    func Encrypt

    func Encrypt(selector, domain string, in []byte) (out []byte, err error)

    Encrypt will encrypt the data given in 'in', and return the encrypted version in 'out', using the public key it finds in the DKIM-like TXT record at <selector>._domainkey.<domain>. Use the same selector in 'Decrypt'
